### after content 

在每个li后面添加一个星星

```html
<style>
    li:after {
        content: "\2605";
    }
</style>
<span>在每个li后面添加一个星星</span>
<ul>
    <li>大飞</li>
    <li>大飞</li>
    <li>大飞</li>
    <li>大飞</li>
    <li>大飞</li>
</ul>
```
